## 🌟 ফিচারসমূহ
- 📊 লাইভ সিগন্যাল (BTC, ETH, BNB, SOL, XRP, ADA, DOGE)
- 👑 VIP সিস্টেম
- 💰 স্টার পেমেন্ট
- 🔗 রেফারেল সিস্টেম
- 🌐 বাংলা ও ইংলিশ সাপোর্ট
- 📰 নিউজ ব্রডকাস্ট

## 🚀 ডিপ্লয়মেন্ট (Railway)

1. গিটহাবে রিপোজিটরি তৈরি করুন
2. Railway তে গিয়ে "Deploy from GitHub" সিলেক্ট করুন
3. আপনার রিপোজিটরি কানেক্ট করুন
4. এনভায়রনমেন্ট ভেরিয়েবল অ্যাড করুন
5. ডিপ্লয় করুন!

## ⚙️ Environment Variables
Railway Dashboard → Variables এ নিচেরগুলো সেট করুন:

| Variable | Value |
|----------|-------|
| `BOT_TOKEN` | আপনার টেলিগ্রাম বট টোকেন |
| `ADMIN_ID` | আপনার টেলিগ্রাম আইডি |
